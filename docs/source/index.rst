欢迎阅读 rcute-ai 的文档
========================

rcute_ai 用来辅助 R-Cute 机器人（如 `Cozmars <https://rcute-cozmars.readthedocs.io>`_ ）完成图像/语音识别等高级功能的 python 模块。
它只是对一些 python 库的简单封装，以提供相对一致的 API
